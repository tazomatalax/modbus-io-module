Polynomial Calibration

Enter polynomial like: 2x^2 + 3x + 1 (where x = raw reading)
Examples:
  • x^2 + 2x + 1 - Quadratic calibration
  • 0.001x^3 - 0.1x^2 + x - Cubic calibration
  • -0.5x^2 + 10x - Parabolic correction

Mathematical Expression

Enter mathematical expression (where x = raw reading)
Examples:
  • x * 1.8 + 32 - Celsius to Fahrenheit
  • (x - 32) / 1.8 - Fahrenheit to Celsius
  • x * x / 100 - Square relationship
  • x + (x * x * 0.001) - Non-linear correction
Allowed operations: +, -, *, /, (), x (raw value), and numbers

State when device boots or resets
Coil address range: 100-200. Master reads/writes pin state here. Write 0 to lock OFF (disable rules). Write ≠0 to unlock.
📋 Coexistence Model - Remote Control & Monitoring
  • Master reads coil: Gets current pin state (0=LOW, 1=HIGH)
  • Master writes 0: Locks pin OFF, disables automation rules
  • Master writes ≠0: Unlocks pin, restores rule control
  • UI unlock button: Also restores rule control if remotely locked
